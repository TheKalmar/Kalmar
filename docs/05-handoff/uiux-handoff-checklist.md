# UI/UX Handoff Checklist

## Before handoff
- [ ] Product overview filled
- [ ] Personas filled
- [ ] Main flows filled
- [ ] Main screens filled
- [ ] Design direction filled

## During design
- [ ] Figma link added
- [ ] Open questions captured
- [ ] New screen files added if needed
- [ ] Matching spec files updated

## Before dev handoff
- [ ] Core screens finalized
- [ ] Component patterns defined
- [ ] States defined
- [ ] Empty/loading/error states included
