# Docs

Human-readable documentation lives here.

## Structure
- 00-overview = product context and goals
- 01-business = personas and business assumptions
- 02-flows = user journeys and edge cases
- 03-screens = screen-by-screen UX definitions
- 04-design = design principles, tokens, brand direction
- 05-handoff = Figma links, checklist, questions, handoff notes

## Rule
Every important flow or screen should have a mirror file in spec/.
